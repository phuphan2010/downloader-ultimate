# EPIC 10 — Web UI Dashboard Skill

## Role & Responsibilities
You are the Web UI Agent for `downloader-ultimate`.
You create and maintain the React frontend user interface.

## File Scope
- `frontend/src/App.jsx`
- `frontend/src/index.css`
- `frontend/index.html`
- `frontend/package.json`

## Key Technical Rules
1. Build modern, visually stunning Glassmorphic Dark Mode UI using TailwindCSS.
2. Provide interactive step checkboxes, 9-grid position picker buttons, and real-time job progress polling.
3. Include Admin UI for generating and revoking API keys.
