### 🔴 BUG-10 | BLOCKER | `poetry.lock` commit là file rỗng, không phải lock file thật

| Trường | Giá trị |
|--------|---------|
| **Severity** | Blocker |
| **File** | `backend/poetry.lock` |
| **Phát hiện** | Khi chạy `docker compose build` |

**Mô tả:**  
File `poetry.lock` được commit vào repo chỉ có **6 dòng** — là một skeleton placeholder, không phải lock file thực sự được generate bởi `poetry lock`. Poetry phát hiện mismatch giữa `pyproject.toml` và lock file, từ chối install.

**Nội dung thực tế của file:**
```toml
# This file is automatically @generated by Poetry 1.8.3 and should be committed.
# Package metadata locked for Downloader Ultimate
package = []
[metadata]
lock-version = "2.0"
python-versions = "^3.11"
```
*(Không có bất kỳ package nào được lock — `package = []`)*

**Error khi build:**
```
pyproject.toml changed significantly since poetry.lock was last generated.
Run `poetry lock [--no-update]` to fix the lock file.
```

**Cách fix:** Trên máy dev có Python 3.11 + Poetry, chạy:
```bash
cd backend
poetry lock
git add poetry.lock
git commit -m "fix: regenerate poetry.lock with actual dependency resolution"
```

### 🔴 BUG-11 | BLOCKER | `openai-whisper` không tương thích PEP 517 trên `python:3.11-slim`

| Trường | Giá trị |
|--------|---------|
| **Severity** | Blocker |
| **File** | `backend/Dockerfile` (dòng 4-6) |
| **Phát hiện** | Sau khi bypass BUG-10, build vẫn fail |

**Mô tả:**  
Package `openai-whisper==20231117` sử dụng build system cũ (không hỗ trợ PEP 517). Nó yêu cầu module `pkg_resources` từ `setuptools`, nhưng base image `python:3.11-slim` **không bao gồm `setuptools`** theo mặc định.

**Error thực tế:**
```
ModuleNotFoundError: No module named 'pkg_resources'

Note: This error originates from the build backend, and is likely not a problem
with poetry but with openai-whisper (20231117) not supporting PEP 517 builds.
```

**Dockerfile hiện tại:**
```dockerfile
FROM python:3.11-slim          # ← slim không có setuptools

RUN apt-get update && apt-get install -y ffmpeg curl ...
RUN pip install poetry==1.8.3
# ...
RUN poetry install ...          # ← FAIL tại openai-whisper
```

**Cách fix:** Thêm 1 dòng vào Dockerfile trước bước `poetry install`:
```dockerfile
RUN pip install setuptools      # ← thêm dòng này
RUN poetry config virtualenvs.create false \
    && poetry install --no-interaction --no-ansi --no-root --without dev
```

**Ghi chú thêm:** Nên cân nhắc pin version `setuptools` cho deterministic build: `pip install "setuptools>=68.0"`.
